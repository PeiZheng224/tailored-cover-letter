# Intake Form

Use this form to collect, extract, or confirm inputs for a tailored U.S.-style cover letter.

## Required Inputs

- Target company name: `[Company]`
- Target role title: `[Role Title]`
- Job description, job posting text, or job posting link: `[JD or Link]`
- Candidate resume, LinkedIn summary, project list, or professional profile: `[Candidate Profile]`

## Optional Inputs

- Preferred career positioning: `[Positioning]`
- Target industry or function: `[Industry or Function]`
- Strongest experiences to highlight: `[Experiences]`
- Weaker or non-obvious match areas: `[Areas]`
- Networking context: `[Employee names, conversation themes, referral context]`
- Recruiter name: `[Recruiter Name]`
- Company office address: `[Address]`
- Target location: `[Location]`
- Preferred tone: `[Tone]`
- Word limit: `[Word Count]`
- Experiences to emphasize: `[Emphasize]`
- Experiences to avoid: `[Avoid]`

## Intake Behavior

If all required inputs are present, proceed directly.

If one or more required inputs are missing, choose the least blocking path:

- If the company or role is missing, ask for it before drafting.
- If the job description is missing but the role is clear, ask for the JD or proceed with clearly labeled assumptions about the role.
- If the candidate profile is missing, ask for a resume, LinkedIn summary, project list, or short professional profile before drafting a candidate-specific letter.
- If only optional fields are missing, proceed and list what the user can personalize later.

## Missing Information Note

Use this format when proceeding with incomplete information:

```markdown
Missing information:
- [Missing input]

Assumptions made:
- [Assumption]

Verify before submission:
- [Item to verify]
```

## Privacy Rule

Do not infer personal details from hidden memory, inaccessible prior conversations, private files, hidden profile data, or unstated context. Use current-invocation information and, when available and permitted, the current agent's visible user profile, memory, or saved preferences.
